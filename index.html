Soy Ivonne Hernández

Conoce a Ivonne Hernández

Compositora y artista sonora. Soy una mente creativa, tengo un gran interés en la creación colaborativa y la vinculación entre arte, ciencia, tecnología y sociedad. Estudié composición en el Conservatorio de las Rosas. Actualmente soy estudiante de la licenciatura de Música y Tecnología Artística en la ENES-UNAM, Unidad Morelia.

Creación Sonora

Desarrollo proyectos sonoros y musicales a partir de paisajes sonoros, grabación de campo, composiciones algorítmicas para la generación de sistemas interactivos.

Reaper SuperCollider Audacity Ableton Live

Media Art

Exploro herramientas tecnológicas para crear arte interactivo, videoarte, instalaciones sonoras.

Hydra Audacity Resolume Arena

Desarrollo Web

Construyo sitios web responsivos y modernos utilizando tecnologías como HTML, CSS, JavaScript y Bootstrap.

HTML CSS JavaScript Bootstrap
